| 脏话     | 替换                                   |
| -------- | -------------------------------------- |
| 卧槽     |                                        |
| 他妈的   |                                        |
| 操他妈的 |                                        |
| 鸡巴、屁 |                                        |
| 装逼     |                                        |
| 傻逼     |                                        |
| 牛逼     | 强，棒，漂亮，厉害、好厉害、超级厉害。 |
