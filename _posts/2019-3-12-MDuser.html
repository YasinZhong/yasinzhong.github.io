--- layout: md title: MarkDown简明用法 --- # MarkDown简明用法 如果你在电脑上写MD，我推荐typora这个开源软件，不需要面对源码。如果你在安卓上使用MD，可能需要手动输入各种代码，有可能需要用到下面这些基本的格式符号，这是一个直接写源码的教程。 ### 标题 在行首插入 1 到 6个#，分别表示标题 1 到标题 6 ``` 格式 ### 这是3号标题 #### 这是4号标题 ``` 输出效果 ### 这是3号标题 #### 这是4号标题 ### 列表、有序列表 有序列表则使用数字加一个英文句点，并加空格，再输入文字 ``` 格式 1. 你好 1. 彩盒科技 1. 加油！ ``` 输出效果 1. 你好 1. 彩盒科技 1. 加油！ ### 无序序列表 无序列表使用星号、加号或是减号作为列表标记，并加一个空格，再输入文字 ``` 格式如下 + Android + iPhone + iPad ``` 输出效果 + Android + iPhone + iPad ### 链接 [描述]（链接地址） > 你可以直接点击工具栏的 link 按钮来快速添加链接! ``` 格式如下 [彩盒科技 ](a.caihe.org) ``` 输出效果 [彩盒科技 ](a.caihe.org) ### 表格 ``` 格式如下 | Tables | Are | Cool | | ------------- |:-------------:| -----:| | col 3 is | right-aligned | $1600 | | col 2 is | centered | $12 | | zebra stripes | are neat | $1 | ``` 输出效果 | Tables | Are | Cool | | ------------- |:-------------:| -----:| | col 3 is | right-aligned | $1600 | | col 2 is | centered | $12 | | zebra stripes | are neat | $1 | ### 支持代码段 ``` int main(){ printf } ```
